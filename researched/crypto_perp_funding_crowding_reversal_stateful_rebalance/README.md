# crypto_perp_funding_crowding_reversal_stateful_rebalance

Researched handoff package built from the fresh new_15 rerun. These candidates
are smoke-screened research outputs, not market validation or live-trading
approval.

This is a frozen archive, not an active validation layout. See `ARCHIVE.md`
before using any variant or evidence file.

## Families

- `families/family_01_primary_time_only_exit`: primary family `time_only_exit`
- `families/family_02_secondary_entry_filter`: secondary family `entry_filter`
- `families/family_03_exploratory_selection_or_breadth`: exploratory family `selection_or_breadth`

Each variant contains a rewritten `config.toml`, the strategy snapshot used by
the rerun, and compact new_15 rerun evidence. Large raw input artifacts are
intentionally omitted and can be recreated by rerunning the config.
